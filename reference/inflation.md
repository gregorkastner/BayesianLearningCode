# Inflation data

Harmonized Index of Consumer Prices (HICP) in the Euro area (changing
composition). Monthly data from January 1997 to June 2025.

## Usage

``` r
inflation
```

## Format

A sequence of length 342.

## Source

European Central Bank Data Portal, retrieved from
<https://data.ecb.europa.eu/data/datasets/ICP/ICP.M.U2.N.000000.4.ANR>,
July 25, 2025.
