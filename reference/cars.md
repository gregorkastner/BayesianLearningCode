# Car Registration Data

Registrations counts of new passenger cars in Austria. Monthly time
series data from January 2000 to January 2026.

## Usage

``` r
cars
```

## Format

A monthly time series object with 313 observations.

## Source

Statistik Austria open.data, retrieved from
<https://data.statistik.gv.at/web/meta.jsp?dataset=OGD_fkfzul0759_OD_PkwNZL_1>,
March 5, 2026.
