
# BayesianLearningCode

<!-- badges: start -->
<!-- badges: end -->

BayesianLearningCode provides R code to reproduce the examples in the
book “Bayesian Learning” by Sylvia Frühwirth-Schnatter, Bettina Grün,
Gregor Kastner, and Helga Wagner.

## Installation

You can install the development version of BayesianLearningCode from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("gregorkastner/BayesianLearningCode")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(BayesianLearningCode)
## basic example code
```
